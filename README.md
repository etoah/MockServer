# MockServer
a quick REST and static server to help the front-end to develop and prototype separate from the back-end progress 

## How to use it
### 1.git clone
``` 
git clone  git@github.com:etoah/MockServer.git
npm install 
nodemon app

```


### 2. npm 

`规划中`
```
npm install mockserverjs  
mockserver init 
mock server start
```
